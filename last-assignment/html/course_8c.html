void enroll_student ( Course *  course,
Student *  student 
)

Adds student to a course by adding one to the total count of students in the course. It then uses calloc to instantiate the list holding students if there are no students enrolled in the course before or uses realloc to resize the list to fit one more student. In the end it adds the student to the list.
